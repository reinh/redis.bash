NAME

redis.bash - a Redis client in Bash

SYNOPSIS

redis.bash ARGS

DESCRIPTION

redis.bash is a Redis client written in (almost) pure bash.

CAVEATS

  • requires a bash with /dev/tcp enabled.
  • does not (yet) handle multi-bulk replies.
  • does not (yet) allow configuration of redis host or port
